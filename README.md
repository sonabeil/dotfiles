# dotfiles

Environment configuration

## Prerequisites

The following things need to be present for this to work:

1. Homebrew
2. Zsh
3. Oh-my-zsh

## Running the script

Help:
```sh
$ ./configure.sh -t h
```

Link files:
```sh
$ ./configure.sh -t build
```

Unlink files:
```sh
$ ./configure.sh -t clean
```
